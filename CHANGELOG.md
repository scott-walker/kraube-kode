## [1.0.0] - 2026-04-06

### Добавлено

- Основной интерфейс чата с поддержкой streaming-ответов
- Inference-блоки: отображение tool use, результатов, approval-запросов
- Diff-блоки с подсветкой синтаксиса
- Copy-to-clipboard для блоков кода
- Боковая панель с навигацией (полная и мини-версия)
- Страница настроек подключений
- Страница общих настроек
- Пул сессий для управления соединениями с Claude Code
- Система тем (светлая/тёмная)
- Дизайн-система на CSS custom properties
- Брендинг: логотип, баннеры
- Архитектура на портах и адаптерах с kraube-konnektor SDK
- Централизованный state management (Zustand)
- SQLite-персистенция для пользовательских настроек
- Сборка для Linux (.deb, .rpm)
